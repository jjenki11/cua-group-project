Theater Project


About Page



The project filters the database for theaters in New York and displays markers on a map overlay.

Clicking on a marker will display a popup with the theater name and a url to open up the theater homepage in another window.

It provides a toggle filter by city: All, New York, Long Island, and Queens.

There is a search bar which can perform partial matching of a theater name.

Lastly there is a zip code search bar which can perform matching of a zip code and only allows numeric input.
